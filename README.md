# Iris Species Data Analysis
An Exploratory Data Analysis (EDA) project focused on identifying patterns in botanical data.

#  Technical Skills Demonstrated
* Python Mastery:* Used `Pandas` for data grouping and `Seaborn` for multi-variable plotting.
* Numerical Aptitude:* Calculated mean, standard deviation, and feature correlations to distinguish between species.
* Communication:*Translated 150 rows of raw data into a clear visual "Pairplot" for easier pattern recognition.

#  Key Insight
The analysis confirms that Iris Setosa is mathematically distinct from other species based on its significantly smaller petal dimensions.
